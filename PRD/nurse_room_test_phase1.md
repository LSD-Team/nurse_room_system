## 3. 💊 ยืมยา — Borrow Medicines (Workflow เต็ม)
> **ลำดับ:** DRAFT → SUBMIT → APPROVE → RECEIVE

### 3.1 สร้างรายการยืม (DRAFT)
- [ ] **3.1.1** เปิดหน้า **ยืมยา** → โหลดรายการยืมทั้งหมดได้
- [ ] **3.1.2** กดปุ่ม "สร้างรายการยืมใหม่" → Dialog/Form เปิดขึ้น
- [ ] **3.1.3** เลือก Supplier + เพิ่มรายการยา → สามารถเพิ่ม/ลบรายการยาได้
- [ ] **3.1.4** บันทึก → สถานะ = **DRAFT**
- [ ] **3.1.5** แก้ไขรายการ DRAFT → แก้ไขและบันทึกได้ สถานะยังเป็น DRAFT

### 3.2 ส่งอนุมัติ (SUBMIT)
- [ ] **3.2.1** กด "ส่งอนุมัติ" จาก DRAFT → สถานะเปลี่ยนเป็น **SUBMITTED**
- [ ] **3.2.2** พยายามแก้ไขหลัง Submit → ไม่สามารถแก้ไขได้

### 3.3 อนุมัติ / ปฏิเสธ (APPROVE / REJECT)
- [ ] **3.3.1** กด "อนุมัติ" → สถานะเปลี่ยนเป็น **APPROVED**
- [ ] **3.3.2** กด "ปฏิเสธ" พร้อมหมายเหตุ → สถานะเปลี่ยนเป็น **REJECTED**
- [ ] **3.3.3** กด "ส่งกลับแก้ไข" (REWORK) → สถานะกลับเป็น **DRAFT**

### 3.4 รับยา (RECEIVE)
- [ ] **3.4.1** กด "รับยา" จากสถานะ APPROVED → สถานะเปลี่ยนเป็น **RECEIVED**
- [ ] **3.4.2** ตรวจสอบ Stock หลังรับยา → `stock_on_hand` เพิ่มขึ้นตามจำนวนที่ยืม
- [ ] **3.4.3** ตรวจสอบ Movement Records → มี record ประเภท `BORROW_RECEIVE`

### 3.5 ยกเลิก (CANCEL)
- [ ] **3.5.1** กด "ยกเลิก" จากสถานะ DRAFT → สถานะเปลี่ยนเป็น **CANCELLED**

## 8. 🗓️ นับสต็อก — Physical Count (Workflow เต็ม)
> **ลำดับ:** สร้าง Period → สร้าง Count → กรอกจำนวน → Submit → Approve

### 8.1 จัดการ Period
- [ ] **8.1.1** เปิดหน้า **นับสต็อก** → แสดงรายการ period ทั้งหมด
- [ ] **8.1.2** สร้าง period ใหม่ → บันทึกสำเร็จ สถานะ = **OPEN**
- [ ] **8.1.3** แก้ไขวันสิ้นสุด period (ยัง OPEN) → อัปเดตได้
- [ ] **8.1.4** ลบ period ที่ยัง OPEN → ลบได้สำเร็จ
- [ ] **8.1.5** ลบ period ที่มี count แล้ว → แสดง error / ป้องกันการลบ

### 8.2 สร้างและกรอก Count
- [ ] **8.2.1** กดเริ่มนับสต็อกใน period → สร้าง count session ได้
- [ ] **8.2.2** หน้ารายละเอียด count แสดงยาทั้งหมด โดย `qty_system` = ค่าตอนเริ่มนับ
- [ ] **8.2.3** กรอก `qty_counted` ทีละรายการ → บันทึก draft ได้ (save-lines)
- [ ] **8.2.4** คอลัมน์ **ส่วนต่าง** คำนวณถูก: `diff = qty_counted - qty_system`

### 8.3 Submit และ Approve
- [ ] **8.3.1** กด "ส่งอนุมัติ" → สถานะเปลี่ยนเป็น **SUBMITTED**
- [ ] **8.3.2** หน้า Approval แสดงรายการที่ SUBMITTED ในคิวอนุมัติ
- [ ] **8.3.3** อนุมัติ count → สถานะ = **APPROVED**, Stock ปรับตาม diff
- [ ] **8.3.4** ตรวจสอบ Stock หลัง Approve → `stock_on_hand` เปลี่ยนแปลงถูกต้อง
- [ ] **8.3.5** ตรวจสอบ Movement Records → มี record ประเภท `ADJUSTMENT`
- [ ] **8.3.6** ปฏิเสธ count (Reject) → สถานะ = **REJECTED**, period กลับสู่ OPEN

### 8.4 รายงานรายเดือน (Stock Monthly Record)
- [ ] **8.4.1** เปิดหน้า **รายงานสต็อกรายเดือน** → แสดงรายการ period ที่ APPROVED
- [ ] **8.4.2** คลิกดูรายละเอียด period → แสดง snapshot ของสต็อก ณ วันที่นับ
